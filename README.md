# Personal_Assistant

## 说明

一个简易的 Google Extensions TodoList，采用 vue.js 开发，使用 IndexedDB 进行数据存储。

目前支持以下功能：
- 新建/编辑/删除/确定完成 todo 项
- 查看正在进行中的 todo 以及已完成、已删除的 todo 项
- 根据项目的优先级/deadline 进行排序
- 查看 todo 详情
- 关键字搜索
- 清空所有数据
